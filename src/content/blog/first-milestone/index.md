This is the first post in the new ROBOSECT blog. From here on, updates about the platform,
hardware milestones, and team news will be posted here as simple Markdown files — easy to
write, easy to version, no separate CMS required.

To publish a new update: add a folder under `src/content/blog/<slug>/` with an `index.md`
file (frontmatter: `title`, `description`, `date`, `cover`, `coverAlt`, `tags`) and a cover
image alongside it, then commit and push.
